# Quarto isomorphic

The game of Quarto, with react in SSR mode.

## Requirements

Make sure to have `nodejs` and `npm` installed.

## Install the game

``` bash
make install
```

## Run the game

Run Quarto isomorphic game

``` bash
make run
```

## Test the code

Run unit tests on source code

``` bash
make test
```

## Lint the code

Inspect source code and writing rules

``` bash
make lint
```
